Layout Grids Overview

The ingrediants for grid

layout container

Using flexbox, .layout is the container for layout columns. Everything about layout exist under this class. A typical grid stucture would look like this

  • .layout
    • .layout__col-modifier
    • .layout__col-modifier

columns

Inside each .layout container, there are resizable columns. They comes with 3 sizes, they will resize according to their .layout size. When the parent size is too small, they will collapse into a single column layout they are:

  • .layout__col-s (small, occupy 1 poportion)
  • .layout__col-m (medium, occupy 2 poportion)
  • .layout__col-l (large, occupy 3 poportion)

Sidebar

The special one here is the "side-bar" column, which is used to define a sidebar that won't grow and shrink when the browser resize. It will hide itself when the viewport is too small, possibily will reveal itself under user interaction(to be implemented).

  • .layout__col-side-bar
